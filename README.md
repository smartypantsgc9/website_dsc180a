# website_dsc180a
Fairly simple setup.

Two command lines.


First command line window:
/> cd website_dsc180 (or whatever directory you will be using)
/> source env/bin/activate
/> python3 app/run.py

Second command line window:

/> cd website_dsc180 (or whatever directory you will be using)
/> source env/bin/activate
/> cd frontend
/> npm start
